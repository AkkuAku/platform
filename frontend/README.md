Как раннить?

cd platform/patient-doctor-platform/frontend
npm install
npm run dev
The app will open at http://localhost:5173


Для тестирования системы пациенты могут войти с такими учетными данными: 
Алексей Иванович Сидоров — логин patient1, пароль 123; 
Ольга Алексеевна Кузнецова — логин patient2, пароль 123; 
Мария Павловна Иванова — логин patient3, пароль 123.

Врачи могут войти с такими учетными данными: 
Иван Иванович Иванов — логин doctor1, пароль 123; 
Мария Алексеевна Петрова — логин doctor2, пароль 123.
